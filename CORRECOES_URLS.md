# ✏️ Correções de URLs - Colabore!

Este arquivo serve para reportar URLs incorretas ou desatualizadas nos portais de transparência.

## 📝 Como contribuir

1. Encontre uma URL incorreta
2. Envie a URL correta neste formato:

```
## [Data] - Nome do Portal

**Portal:** Nome do portal
**URL Atual (incorreta):** https://url-antiga.gov.br
**URL Correta:** https://url-nova.gov.br
**Estado:** UF
**Reportado por:** @seu-usuario
```
