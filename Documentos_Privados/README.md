## 🔄 Como Fazer Backup

### Opção 1: Script Automático
```
Duplo clique: scripts\backup-diario.bat
```

### Opção 2: Git
```bash
cd Documentos_Privados
git add .
git commit -m "Backup %date%"
git push origin main
```

### Opção 3: Manual
```
Copiar pasta para:
- HD Externo
- Pen drive
- Outro PC
```

## 🔐 Dicas de Segurança

1. **Nunca** deixe senhas no mesmo lugar que documentos
2. **Use** criptografia para dados sensíveis
3. **Verifique** backups periodicamente
4. **Tenha** cópia em 2 lugares diferentes
